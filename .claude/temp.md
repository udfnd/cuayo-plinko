해당 repo에 지금 연결되어 있는 supabase를 활용해서 다음과 같은 기능을 가지고 있는 회원가입/로그인 및 회원 기능을 제작하고 싶어.

1. 회원가입 기능
   - 이메일, 비밀번호, 닉네임 입력
   - 비밀번호 암호화 저장
   - 기본적인 유효성 검사 (이메일 형식, 비밀번호 강도 등)
   - 중복된 이메일 및 닉네임 검사
   - 기타 회원가입 시에 필요한 기능들 추가

2. 로그인 기능
   - 이메일, 비밀번호 입력
   - 세션 관리 (JWT 토큰 발급 등)
   - 로그인 시도 제한 (brute-force 공격 방지)
   - 비밀번호 재설정 기능
   - 기타 로그인 시에 필요한 기능들 추가

3. 잔고 조회 및 관리 기능
   - 회원의 잔고 조회 및 회원의 잔고에 따른 게임 플레이 가능
   - 회원가입 시 초기 잔고 1000원 지급
   - 잔고가 0원 이하일 경우 "관리자에게 잔고 충전 요청" 버튼 출력, 및 버튼 클릭 시 "도박 중독은 1336" 안내 문구 출력

또한 현재 게임을, 로그인한 사용자 전체가 참여할 수 있는 멀티플레이 형식으로 제작하고 싶어. 따라서 holdem exchange 게임도 crush 게임과 비슷하게 자동으로 진행되게 하고 싶어. 현재 crush 게임과 holdem exchange 게임을 접속한 사용자들이 공유할 수 있는 구조로 코드를 수정해주고, holdem exchange 게임은 각 단계마다 10초 간격을 두고 진행되도록 코드를 수정해줘.
