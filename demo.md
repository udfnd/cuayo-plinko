# Plinko Demo 시연 스크립트

이 문서는 1~2분 분량의 데모 시연 스크립트입니다.

---

## 시연 순서

### 1. 프로젝트 소개 (10초)

> "안녕하세요. 지금부터 Plinko 데모 웹앱을 보여드리겠습니다.
> 이 프로젝트는 교육/시연 목적으로 제작되었으며, 실제 도박 서비스가 아닙니다."

### 2. 화면 구성 설명 (20초)

> "화면은 크게 세 부분으로 나뉩니다:
> - 왼쪽: 설정 패널 (베팅금액, 행 수, 리스크, 시드)
> - 가운데: Plinko 보드 (핀과 슬롯)
> - 오른쪽: 결과 패널 (배수, 수익, 확률 분포)"

### 3. 기본 드롭 시연 (20초)

> "먼저 기본 설정으로 공을 떨어뜨려 보겠습니다."

**동작:**
1. 기본 설정 확인 (Rows: 12, Risk: Medium, Bet: 1.00)
2. **Drop Ball** 버튼 클릭
3. 공이 핀을 통과해 슬롯에 떨어지는 애니메이션 관찰
4. 결과 확인 (슬롯 번호, 배수, 지급액)

> "공이 핀을 통과하며 좌우로 튕기고, 최종적으로 슬롯에 떨어집니다.
> 배수에 따라 지급액이 계산됩니다."

### 4. 리스크 레벨 비교 (20초)

> "리스크 레벨에 따른 배수 차이를 보여드리겠습니다."

**동작:**
1. Risk를 **Low**로 변경 → 확률 테이블 관찰 (가운데 안정적)
2. Risk를 **High**로 변경 → 확률 테이블 관찰 (극단 배수 증가)

> "Low는 안정적이고, High는 극단적입니다.
> 오른쪽 확률 테이블에서 각 슬롯의 배수와 확률을 확인할 수 있습니다."

### 5. 시드 기반 재현성 (20초)

> "시드를 이용해 결과를 재현할 수 있습니다."

**동작:**
1. Seed 입력란에 `test-seed-1` 입력
2. Drop 3회 → 결과 기억
3. 페이지 새로고침
4. 같은 시드 입력 후 Drop 3회 → 동일 결과 확인

> "같은 시드에서는 항상 같은 결과가 나옵니다.
> 이는 검증이나 디버깅에 유용합니다."

### 6. Auto 모드 시연 (15초)

> "Auto 모드로 연속 드롭을 할 수 있습니다."

**동작:**
1. Auto 횟수를 10으로 설정
2. **Auto (10×)** 버튼 클릭
3. 다중 공이 떨어지는 모습 관찰
4. 히스토리에 결과가 쌓이는 것 확인

> "히스토리에서 최근 20개 결과를 확인할 수 있고,
> 총 수익도 실시간으로 계산됩니다."

### 7. 마무리 (10초)

> "이상으로 Plinko 데모 시연을 마치겠습니다.
> 기대값(EV)은 0.99로 설정되어 있어 1% 하우스 엣지가 반영되어 있습니다.
>
> 다시 한번 강조하면, 배수표는 독자적으로 설계된 값입니다.
> 감사합니다."

---

## 핵심 포인트 요약

1. **교육/데모 목적** - 실제 도박 서비스 아님
2. **독자 배수표** - 상용 서비스와 동일하지 않음
3. **시드 재현성** - 같은 시드 = 같은 결과
4. **확률 투명성** - 이항분포 기반 정확한 확률 표시
5. **EV = 0.99** - 1% 하우스 엣지 반영
